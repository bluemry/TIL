# 애플리케이션 계층의 역할
  ## 애플리케이션 계층 : 사용자가 직접 사용하며 체감하는 계층
  - TCP/IP 프로토콜 계층중, 서비스 영역은 여기가 유일하고 나머지는 다 통신.
  - 간접적으로 사용하는 프로토콜 (DNS, DHCP, SSL/TLS, NTP, LDAP)

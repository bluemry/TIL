# 시간복잡도

## 배경

: 프로그램의 성능을 측정하려면?

- 입력 크기
- 하드웨어 성능
- 운영체제 성능
- 컴파일러 최적화
- 비동기 로직 등등..

> 결론은 정확히 파악하긴 불가능이고,
> 상대적, 대략적 파악하는 방법이 "빅오표기법".

- 실수 < 지수 < 선형 < 지수\*선형 < 선형제곱
- 빅오끼리는 곱, 합의 법칙
- 2가지만 기억하기 : 가장 큰 항 기준
- 자바스크립트 성능 측정 방법 : Date객체 사용
  ex)
  const start = new Date().getTime();
  ~코드
  const start = new Date().getTime();
  console.log(end - start);
  (밀리초.)
